# SAWWiki

Welcome to SAWWiki, a wiki site and a product of [sawinfotech.com](https://www.sawinfotech.com). 

Live site: https://sawwiki.sawinfotech.com/

The wiki application presents content with a simple editor workflow, page preview, revision metadata (changed by, version, hit count), history view, and support for attachments. It is designed for teams to maintain documentation collaboratively with role-based access for readers, editors, and administrators.

## Tech Stack

- ASP.NET Web Forms (.aspx, code-behind)
- C# on .NET Framework 3.5
- ADO.NET (SqlConnection/SqlDataAdapter) for data access
- Microsoft SQL Server (stored-procedure driven page content)
- HTML, CSS, JavaScript for UI and wiki-editing helpers
- Uses ASP.NET membership and web parts features.
